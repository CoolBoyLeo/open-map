# 用户贡献数据说明

欢迎参与 Open Map 开放地图平台的数据共建！为了保证数据质量和一致性，请按照以下规范准备和提交数据。

---

## CSV 文件格式规范

上传的数据应为 CSV 格式，必须包含以下字段：

| 字段名       | 说明                         | 示例                             |
| ------------ | ---------------------------- | -------------------------------- |
| `name`       | 数据分类名称（统一归属名称） | 商场免费停车                     |
| `icon`       | 该分类对应的系统 Emoji 图标  | 🅿️                              |
| `address`    | 具体地点地址                 | 深圳福田卓悦汇购物中心           |
| `description`| 对该地点的补充说明（可选）   | 任意消费并积分可获赠1张2h停车优惠券 |
